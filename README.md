# TaxBreakPolicyEffect
Main dataset can be download from https://www150.statcan.gc.ca/n1/pub/71m0001x/71m0001x2021001-eng.htm, which is Canada LFS's public use micro file.
CPI.xlsx is used to deflate wages across years, alternatively, you can use year fixed effect to do so.
The Jupyter notebook file involves introductory analysis of the impact of this tax break policy on labor demand.
